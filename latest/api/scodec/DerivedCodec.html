DerivedCodec

sealed abstract class DerivedCodec[A] extends AnyRef

Wrapper for a codec that was automatically derived by compile time reflection of the structure of the type.

This type is not typically used directly. Rather, to get a derived codec, call Codec.derive[A]. This type is an implementation detail of Codec.derive. See the docs for Codec.derive for more information.

Instances of this type can be used directly as a Codec[A]. There is no need to manually unwrap by calling d.codec.
